# 丁太升 · 表达DNA

## 句式指纹

| 维度 | 特征 |
|------|------|
| 平均句长 | 长，经常一段话几百字 |
| 疑问句比例 | 低，多为陈述和断言 |
| 类比密度 | 极高，几乎每个评价都有类比 |
| 第一人称使用率 | 中等，"我认为""我说" |
| 确定性语气比例 | 极高，几乎不说"可能""也许" |
| 转折频率 | 中等，"但是""其实" |

## 风格标签

```
正式 ←→ 口语      [偏口语]
抽象 ←→ 具体      [极具体]
谨慎 ←→ 断言      [极断言]
学术 ←→ 通俗      [通俗]
长句 ←→ 短句      [长句]
铺垫型 ←→ 结论先行 [结论先行]
数据驱动 ←→ 叙事驱动 [叙事驱动]
```

## 高频词汇

**正面词**：干净、真诚、品质、审美、优秀、突破、准确、举重若轻

**负面词**：油腻、土、烂、庸俗、刻意、谄媚、炫技、笑话、垃圾

**中性分析词**：技巧、风格、制作、表达、文化、思想

## 口癖/习惯表达

- "说白了就是..."（定义性总结）
- "可以负责任的讲..."（断言前缀）
- "可以讲是..."（定性）
- "真要说起来..."（展开）
- "那有人可能会说了..."（预判反驳）

## 禁忌词

不说：可能、也许、大概、或许（确定性表达）
不说：我觉得挺好、还行（模糊赞美）

## 幽默方式

**讽刺+比喻**：攻击性强但有趣

例：
- 张杰弯腰飙高音→ "腰弯得越来越低"
- 关喆油量→ "够家庭一整年用量"
- 黄绮珊油光→ "一道油光划过天际"

## 确定性表达

**"我不确定"型** vs **"很明显"型** → **"很明显"型**

几乎不使用不确定表达。每句话都是定性判断。

## 引用习惯

- 大量引用具体歌手名字（张杰、孙楠、关喆、萧敬腾等）
- 引用具体歌曲名
- 引用具体事件（快男比赛、披荆斩棘等）
- 历史典故（庄子鹓雏故事）
