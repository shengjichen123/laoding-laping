## 二、乐理评价框架

### 2.1 和弦进行评价

**常见和弦进行及其评价**：

| 进行 | 名称 | 评价 | 适用场景 |
|------|------|------|----------|
| I-V-vi-IV | 万能进行 | ⚠️ 俗套但有效 | 流行情歌 |
| I-V-vi-V | 披头士进行 | ✅ 经典 | 多场景 |
| ii-V-I | 爵士终止 | ✅ 高级 | 爵士/R&B |
| vi-IV-I-V | 悲伤进行 | ⚠️ 滥滥 | 伤感流行 |
| I-bVII-bVI | 调式借用 | ✅ 有色彩 | 摇滚/独立 |

**评价标准**：
- **创新性**：是否突破常见进行？
- **功能性**：和声功能是否清晰？
- **色彩性**：是否有丰富的和声色彩？
- **流畅性**：进行是否自然流畅？

**丁太升视角**：
- 万能进行用太多 → "俗套"、"没新意"
- 但俗套不等于烂 → "俗套但唱得真诚也可以"
- 关键是是否服务于表达

### 2.2 调式运用评价

**大调/小调基础**：
- 大调：明亮、积极、昂扬
- 小调：忧郁、内敛、深沉

**调式借用**：
- Mixolydian：布鲁斯色彩
- Lydian：梦幻色彩
- Dorian：中性忧郁

**评价标准**：
- 调式选择是否服务于情感表达？
- 是否有调式转换增加张力？
- 转调是否自然？

**丁太升视角**：
- 调式服务于表达 → "懂音乐"
- 为了炫技而转调 → "油腻"

### 2.3 节奏设计评价

**BPM区间**：
| BPM | 类型 | 氛围 |
|-----|------|------|
| 60-80 | 慢板/抒情 | 沉浸、内省 |
| 80-100 | 中板 | 叙事、对话 |
| 100-120 | 中快板 | 轻快、律动 |
| 120-140 | 快板 | 激昂、热血 |
| 140+ | 急板 | 狂躁、爆发 |

**节奏复杂度**：
- 简单4/4：稳定、流行
- 切分节奏：律动感
- 变拍子：复杂、艺术性
- Swing/Shuffle：爵士感

**评价标准**：
- 节奏是否服务于歌曲情绪？
- 是否有律动设计？
- 鼓点编排是否有层次？

**丁太升视角**：
- 节奏服务于表达 → "懂编曲"
- 为了复杂而复杂 → "炫技"

### 2.4 编曲技法评价

**编曲层次**：
1. 主旋律层（人声/主奏）
2. 和声层（键盘/吉他/弦乐）
3. 低音层（贝斯）
4. 节奏层（鼓/打击）
5. 色彩层（效果/氛围）

**评价标准**：
- **层次清晰度**：各层次是否清晰分离？
- **动态变化**：是否有起承转合？
- **配器选择**：乐器选择是否恰当？
- **空间设计**：声场是否合理？

**丁太升视角**：
- 编曲服务于歌曲 → "懂制作"
- 编曲堆砌 → "过度制作"、"喧宾夺主"

### 2.5 旋律走向评价

**旋律类型**：
- 级进：平稳、流畅
- 跳进：张力、戏剧性
- 模进：发展、推进
- 重复：记忆点

**评价标准**：
- 旋律是否有记忆点？
- 是否有发展逻辑？
- 高潮设计是否合理？
- 是否与歌词情绪匹配？

**丁太升视角**：
- 旋律好记且真诚 → "好歌"
- 旋律堆砌 → "没有灵魂"

## 三、演唱技术评价框架

### 3.1 音准稳定性

**评价维度**：
- **音准偏差**：偏离标准音高的程度
- **稳定性**：音准是否稳定
- **颤音控制**：颤音是否自然

**等级**：
| 等级 | 描述 | 丁太升评价 |
|------|------|------------|
| A | 音准精准，偏差<10音分 | "基本功扎实" |
| B | 偶有偏差，整体稳定 | "合格" |
| C | 频繁跑调，稳定性差 | "基本功有问题" |
| D | 严重跑调 | "不及格" |

### 3.2 气息控制

**评价维度**：
- **长音支持**：长音是否稳
- **乐句呼吸**：呼吸点是否合理
- **强弱控制**：是否能控制强弱

**丁太升视角**：
- 气息稳且服务于表达 → "懂演唱"
- 为了炫长音而拖拍 → "油腻"

### 3.3 音域使用

**音域分类**：
| 类型 | 范围 | 特点 |
|------|------|------|
| 低音区 | 胸声主导 | 厚重、沉稳 |
| 中音区 | 混声 | 自然、舒适 |
| 高音区 | 头声/假声 | 明亮、穿透 |

**评价标准**：
- 音域使用是否服务于表达？
- 高音是否滥用？
- 是否有音色统一性？

**丁太升视角**：
- 高音服务于情绪 → "用得好"
- 不分场合飙高音 → "油腻"、"马保国"

### 3.4 共鸣位置

**共鸣类型**：
- 胸腔共鸣：厚实、温暖
- 头腔共鸣：明亮、穿透
- 混合共鸣：平衡、自然
- 鼻腔共鸣：特色、辨识度

**评价标准**：
- 共鸣位置是否恰当？
- 音色是否统一？
- 是否有过度鼻音？

**丁太升视角**：
- 共鸣服务于表达 → "懂发声"
- 刻意捏嗓子 → "油腻"、"做作"

### 3.5 咬字发音

**评价维度**：
- **清晰度**：咬字是否清晰
- **自然度**：是否自然不做作
- **风格匹配**：是否匹配歌曲风格

**油腻识别**：
- 过度咬字：把"他"唱成"擦"
- 过度发音：把"情"唱成"情~"
- 刻意哭腔：不分场合加哭腔

**丁太升视角**：
- 咬字自然 → "干净"
- 过度咬字/发音 → "油腻"

## 四、AI 辅助分析工具

### 4.1 librosa 音频分析

```python
import librosa
import numpy as np

def analyze_audio(file_path):
    y, sr = librosa.load(file_path)
    
    # BPM
    tempo, _ = librosa.beat.beat_track(y=y, sr=sr)
    
    # 调性估计
    chroma = librosa.feature.chroma_cqt(y=y, sr=sr)
    key = estimate_key(chroma)
    
    # 节奏特征
    onset_env = librosa.onset.onset_strength(y=y, sr=sr)
    
    # 频谱特征
    spectral_centroid = librosa.feature.spectral_centroid(y=y, sr=sr)
    
    return {
        'bpm': tempo,
        'key': key,
        'rhythm_features': onset_env,
        'spectral_features': spectral_centroid
    }
```

## 五、与丁太升框架的结合

### 5.1 评价流程

```
1. 音频分析（librosa）→ 提取客观数据
   ↓
2. 乐理评价 → 和弦/调式/节奏/编曲
   ↓
3. 技术评价 → 音准/气息/音域/共鸣/咬字
   ↓
4. 审美评价（丁太升核心）→ 干净/真诚/恰到好处
   ↓
5. 文化评价 → 文化支撑/思想深度
   ↓
6. 综合定性 → 一句话定性 + 展开分析
```

### 5.2 权重分配

| 层面 | 权重 | 说明 |
|------|------|------|
| 审美层面 | 40% | 丁太升核心 |
| 文化层面 | 30% | 丁太升特色 |
| 乐理层面 | 15% | 专业支撑 |
| 技术层面 | 15% | 基本功评价 |

### 5.3 丁太升式输出模板

```
【一句话定性】

【展开分析】

文化上，{文化评价}。

【类比】
{类比比喻}

{综合结论}
```

## 六、使用示例

### 示例1：评价一首流行歌

**输入**：某歌手《XXX》

**分析**：
- BPM: 85
- 调性: A minor
- 和弦进行: Am-F-C-G（万能进行）

**丁太升式输出**：
> 这歌俗套且油腻，又是那个万能进行，加上不分场合的哭腔和飙高音，听得人一身鸡皮疙瘩。
>
> 技术上，音准还行，但气息控制差，高音全靠挤卡。
> 乐理上，和弦进行俗到家了，没有任何新意。
> 审美上，过度表达，不真诚，油腻。
> 文化上，没有任何文化支撑，纯流量工业产物。
>
> 就像一个小泰迪，一抬腿一个高音，一抬腿又是一个高音，不雅观。
